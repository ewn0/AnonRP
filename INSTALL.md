# Installation des dépendances Pusher

Dans le terminal (au dossier du projet), tape :

```bash
npm install pusher pusher-js
npm install --save-dev @types/node
```

- `pusher` : SDK Pusher côté serveur Node.js
- `pusher-js` : SDK Pusher côté client (navigateur)

Après l'install, applique le patch puis redémarre `npm run dev`.
